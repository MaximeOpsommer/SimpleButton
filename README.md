Nous avons été autorisé à réaliser les TPs en binome, nous aurons donc le même rendu.

Pour ce TP, toutes les questions ont été traitées sauf la dernière concernant les modificateurs.
Lors du lancement, 2 fenêtres s'ouvriront :
	- une contenant le bouton interactif
	- l'autre affichant la machine à état qui découle de ce bouton
	
Il est possible de choisir le bouton de la souris à utiliser pour déclencher les actions.
Pour cela, il faut passer en argument du programme :
	1	pour le clic gauche
	2	pour le clic central
	3	pour le clic droit

Si aucun argument n'est précisé ou que ce dernier est incorrect, le clic gauche est choisi par défaut
